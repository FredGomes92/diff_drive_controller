# Autonomous navigation using ROS Navigation Stack

## Parts:

- Raspberry PI
- 2x Arduino nano (one for each motor, the communication with the ROS master is done trough I2C)
- Motor encoders
- Robot chassis
- Boost converter (to provide 6V to the motors)
